# QA Scenarios

## Tenancy and Access
- Owner from Org A cannot read students from Org B.
- Parent can only view feed posts for linked students.

## Attendance
- Kiosk tap cycles status correctly.
- List mode status changes persist by session key.
- Save/load session restores expected statuses.

## Moments + Feed
- Capture draft stores student/session tags.
- Feed draft generation returns exactly 3 sentences.
- Tutor can edit and publish draft without validation errors.

## Billing
- Session credit entry reduces due amount.
- Invoice summary totals match ledger math.
- Credit applies to next invoice period correctly.
